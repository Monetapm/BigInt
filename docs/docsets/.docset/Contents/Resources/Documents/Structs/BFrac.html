BFrac

public struct BFrac : CustomStringConvertible, Comparable, Equatable

A signed fraction with numerator and denominator of unbounded size. A BFrac value is represented by a numerator and a denominator that are BInt’s. The representation is normalized, so that numerator and denominator has no common divisors except 1. The denominator is always positive, 0 has the unique represenentation 0/1
